### Roadmap du Projet
## Voici l'évolution prévue pour AllDrawIO.

## Phase 1 : Fondations & Temps Réel (Terminé)

[x] Infrastructure de base : Serveur Node.js avec Express.

[x] Communication Temps Réel : Intégration de Socket.io pour l'échange de données.

[x] Moteur de dessin : Gestion du HTML5 Canvas (tracés fluides, arrondis).

[x] Synchronisation multi-utilisateurs : Voir les dessins des autres en direct.

[x] Gestion de l'historique (Persistance) : Les nouveaux arrivants voient le dessin déjà existant.

## Phase 2 : Interface & Outils (En cours)

[ ] Palette de couleurs : Sélectionneur de couleurs (Noir, Rouge, Bleu, Vert, etc.).

[ ] Épaisseur du trait : Slider pour ajuster la taille du pinceau.

[ ] Outil Gomme : Mode effaçage pour corriger des zones précises.

[ ] Bouton "Clear All" : Nettoyage complet du tableau (synchronisé sur le serveur).

[ ] Indicateurs de présence : Voir le nombre d'utilisateurs connectés.

## Phase 3 : Fonctionnalités Avancées (À venir)

[ ] Curseurs distants : Voir la position de la souris des autres utilisateurs avec leur pseudo.

[ ] Export d'image : Bouton pour télécharger le chef-d'œuvre au format .png.

[ ] Système de "Rooms" : Possibilité de créer différents salons privés pour dessiner à plusieurs groupes.

[ ] Formes géométriques : Outils pour tracer des lignes droites, des cercles et des rectangles.

[ ] Chat textuel : Un petit espace de discussion à côté du tableau.
